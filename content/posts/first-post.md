Welcome to my new blog! This site is powered by Tina CMS and Next.js.

## What is Tina CMS?

Tina is a fully open-source headless CMS that allows you to edit your content directly on your site. It's built for modern web frameworks like Next.js.

## Features

- **Visual Editing**: Edit your content with a beautiful visual editor
- **Git-backed**: All content is stored in your repository
- **Markdown Support**: Write your posts in markdown
- **Developer Friendly**: Built with developers in mind
